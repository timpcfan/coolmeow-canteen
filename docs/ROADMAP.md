# 冷喵食堂 Roadmap

## V2（智能化增强）

1. 更细粒度营养约束
- 支持热量、蛋白质、脂肪、碳水目标
- 儿童/成人差异化人份规则

2. 计划质量提升
- 引入多目标优化（成本、营养、烹饪时长）
- 自动避免重复菜品组合
- 支持工作日/周末不同策略

3. 采购与库存增强
- 单位换算与包装规格推荐（如“买 500g 装 2 包”）
- 临期优先消耗策略
- 采购预算上限与比价接口

4. 交互增强
- 一键“本餐开做”模式（实时步骤计时）
- 拖拽调整甘特步骤

## V3（平台化与 iOS 迁移）

1. 多端同步
- 账户体系与云同步
- 家庭成员协作（共享计划/购物清单）

2. iOS 迁移
- 复用规则引擎与数据接口
- React Native 或 SwiftUI 前端实现

3. 智能推荐进阶
- 基于历史行为的个性化菜谱排序
- 季节性与本地食材可得性融入推荐

4. 生态连接
- 接入外卖/生鲜平台 API 自动下单
- 设备联动（智能电饭煲/蒸锅）
