# 冷喵食堂（CoolMeow Canteen）PWA V1

冷喵食堂是一个面向家庭用户的餐食规划 PWA，目标是帮助用户在一周维度内高效、节约地做饭。

## 技术栈

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS
- SQLite + Drizzle ORM
- PWA（manifest + service worker）
- Node 原生测试运行器（`node:test`，关键逻辑单测）

## 功能清单（V1）

- 周计划页：
  - 一键生成 7 天早餐/午餐/晚餐
  - 手动替换某餐
  - 每餐展示主食/菜品/汤结构
  - 每餐显示烹饪步骤甘特图（工具并发）
- 库存页：
  - 食材库存增删改查
  - 保质期剩余天数显示
- 采购清单页：
  - 按缺口汇总采购项
  - 输出最晚需要时间与建议采购日期
- 偏好设置页：
  - 人份、烹饪方式、口味、忌口、过敏原
  - 低卡偏好
  - 是否需要汤
  - 主食是否必含
- 工具管理页：
  - 工具类型、数量/层数（并发能力）

## V1.1 体验增强三件套

1. 新手引导 + 一键默认配置
- 首次进入周计划页自动弹出 3 步引导（人数、是否需要汤、常用工具）
- 完成引导后自动写入偏好和工具配置，并立即生成一周计划
- 支持跳过引导；在偏好页支持重新进入引导

2. 周计划工作台快捷调优
- 每餐新增三个快捷操作：
  - 换成更快（优先总时长短）
  - 换成更省钱（优先库存命中和低成本食材）
  - 换成更低卡（优先低热量组合）
- 保留原手动替换入口
- 操作后该餐与甘特图立即刷新

3. 开饭倒计时模式（倒排甘特）
- 周计划页新增入口：“我想在 XX:XX 开饭”
- 可选择当前餐进行倒排，自动计算建议开始时间
- 甘特图高亮“当前应开始步骤”和“可并行步骤”

## 快速开始

```bash
npm install
npm run dev
```

默认地址：`http://localhost:3000`

## 数据初始化

- 应用首次启动时会自动检测空库并注入本地种子数据。
- 也可以手动重置并写入种子：

```bash
npm run db:seed
```

SQLite 文件位置：`db/sqlite.db`

## 常用命令

```bash
npm run dev
npm run lint
npm run test
npm run db:seed
```

## V1.1 使用方式

1. 首次打开应用时，在周计划页完成 3 步引导；或点击“跳过引导”先体验。
2. 在周计划卡片中点击“换成更快 / 更省钱 / 更低卡”快速调优该餐。
3. 在周计划顶部输入目标开饭时间，选择当前餐后点击“开始倒排”进入倒计时甘特。
4. 在偏好设置页底部可使用“标记为跳过引导”和“重新进入引导”入口。

## 核心设计概览

- 规则引擎输入：人份、偏好、库存、工具能力、菜谱库
- 输出：
  - 周计划（每餐主食/菜品/汤）
  - 并发调度甘特图任务
  - 采购缺口与采购时点建议
- 替换逻辑：支持单餐重算（排除当前组合优先）

## 目录结构

```text
app/                 # 页面与 API 路由
components/          # 通用 UI 组件（导航、甘特图、PWA 注册）
db/                  # Drizzle schema、初始化、种子
docs/                # 架构与路线图
lib/                 # 仓储层、规划引擎、序列化
tests/               # 单元测试
public/              # PWA 资源（manifest/sw/icon）
```

## 说明

- V1 采用规则引擎，不涉及机器学习模型。
- 单位换算暂按同单位匹配，跨单位转换将在后续版本扩展。
